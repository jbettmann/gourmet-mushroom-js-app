# PokeDex App

## Description:

This PokeDex is a small web application build with HTML, CSS, and
JavaScript that loads data from an external API \([Pok√©API](https://pokeapi.co)\). Once the data from server has been loaded, user are able view their favorite pokemon and details of pokemon by clicking on the CSS styled pokemon button. A modal, built with Bootstrap, is then displayed with the Pokemon's name, height and type(s). User can also search for a specific Pokemon via the search bar at the top right corner of the PokeDex.

## Key Features

- Load data from an external source (API)
- View a list of items
- On user action (e.g., by clicking on a list item), view details for that item

## Technical Requirements

- The app must load data from an external API.
- App must display a list of items loaded from that API after the page is loaded.
- Must allow viewing of more details for a given list item when clicking on.
- The app must have CSS styling.
- JavaScript code must be formatted according to ESLint rules.
- The app must use at least one additional complex UI pattern, such as a modal, for details or touch interactions.
- Allows searching for items.
- The app must work in Chrome, Firefox, Safari and Edge.
