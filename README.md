# PokeDex App

## Description:

This PokeDex is a small web application build with HTML, CSS, and
JavaScript that loads data from an external API \([Pok√©API](https://pokeapi.co)\). Once the data from server has been loaded, user are able view their favorite pokemon and details of pokemon by clicking on the CSS styled pokemon button. A modal, built with Bootstrap, is then displayed with the Pokemon's name, height and type(s). User can also search for a specific Pokemon via the search bar at the top right corner of the PokeDex.
